## Calculator APP

**M+**：表示将显示值加入存储器中。

**M-**：表示从存储器的数值中减去显示值后并重新存储。

**M**：表示将存储器的数值显示到屏幕。

源码：[QT TUTORIAL 2 : C++ CALCULATOR](http://www.newthinktank.com/2018/06/qt-tutorial-2-c-calculator/)
